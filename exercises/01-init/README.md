# 🎮 تمرین عملی فصل ۱
<div dir="rtl">

**📌 مرحله ۱: نصب گیت**
گیت رو روی سیستمت نصب کن و با git --version تست کن


**📌 مرحله ۲: تنظیمات**

نام و ایمیلت رو تنظیم کن


**📌 مرحله ۳: اولین مخزن**

</div>


```bash
# یه پوشه برای تمرین بساز
mkdir git-practice
cd git-practice

# تبدیل به مخزن گیت
git init

# ببین پوشه .git ساخته شده؟
ls -la

# یه فایل تست بساز
echo "این اولین قدم منه" > test.txt

# ببین گیت چه وضعیتی داره
git status
# می‌بینی که test.txt قرمزه؟ یعنی گیت می‌بینه‌ش ولی track نمیکنه
```

**📌 مرحله ۴: یه کم کنجکاوی کن!**
